# Smart Travel Planning Assistant
### CP3406 Mobile Computing – Assignment 

This Android app helps users plan and organize trips efficiently by integrating:
- Booking management
- Expense tracking
- Daily itinerary builder
- Collaboration and reminders

## Tech Stack
- **Language:** Kotlin
- **UI Framework:** Jetpack Compose
- **Database:** Room / Firebase
- **Version Control:** Git & GitHub
